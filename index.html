Checklist de Validação SFMC

Validação para Criação de Data Extensions

Verificação de nome, estrutura, configurações de retenção e acessos para Data Extensions.

Nome da Data Extension segue a convenção padrão
Armazenada na pasta correta
Descrição preenchida com o objetivo e informações importantes
Chave primária configurada para evitar duplicidade
Campos configurados com tipos de dados corretos
Verificação do tamanho adequado dos campos de texto
Política de retenção configurada de acordo com diretrizes
Permissões de acesso configuradas corretamente
Campos frequentemente usados em filtros foram indexados
Fonte de dados validada e documentada
Importação de dados testada para validar comportamento da DE
